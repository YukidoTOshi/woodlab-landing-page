# WoodLab - Landing Page

Landing page desenvolvida para uma empresa fictícia de móveis planejados.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Layout responsivo
- Galeria de projetos
- Botões de orçamento via WhatsApp
- Navegação por seções

## Objetivo

Projeto criado para prática de desenvolvimento front-end e composição de portfólio.
